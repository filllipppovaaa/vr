Интерактивный 3D Робот

Демонстрация работы с 3D-моделью robot.glb. Используются две техники анимации: автозапуск и плавные переходы между клипами.

Имя файла: robot.glb
Тип: 3D Object
Размер: 98 КБ
Дата изменения: 10.02.2026 22:01

Автозапуск анимации

Модель автоматически начинает анимацию при загрузке благодаря атрибуту autoplay. Анимация будет воспроизводиться непрерывно, пока вы не поставите её на паузу.

<model-viewer src="robot.glb" autoplay animation-name="All Animations">

Переходы между анимациями

Плавные переходы между анимациями с помощью атрибута animation-crossfade-duration="1000". Каждая анимация плавно перетекает в следующую в течение 1 секунды.

<model-viewer src="robot.glb" autoplay animation-crossfade-duration="1000">

Инструкция по использованию

1. Сохраните этот код как файл index.html в той же папке, где находится robot.glb

2. Откройте файл index.html в браузере (двойной клик)

3. Если у вас есть файл poster.webp, поместите его в ту же папку для отображения постера

4. Используйте кнопки под каждой моделью для управления анимациями

5. Для AR-режима на мобильном устройстве нажмите значок AR в правом нижнем углу модели
