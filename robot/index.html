Интерактивный 3D Робот

Демонстрация работы с 3D-моделью robot.glb. Используйте кнопки для управления анимациями.

Автозапуск анимации

Модель автоматически начинает анимацию при загрузке благодаря атрибуту autoplay.

<model-viewer src="robot.glb" autoplay animation-name="All Animations">

Переходы между анимациями

Плавные переходы между анимациями с помощью animation-crossfade-duration="1000".

<model-viewer src="robot.glb" autoplay animation-crossfade-duration="1000">

Инструкция по использованию

1. Сохраните как index.html в папке с robot.glb

2. Откройте index.html в браузере

3. Используйте кнопки для управления анимациями

4. Для AR-режима на мобильном нажмите значок AR
